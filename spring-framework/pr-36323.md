## üéì What I Learned

### Technical Learning

#### 1. AsciiDoc Tab Syntax
```asciidoc
[tabs]
======
Java::
+
[source,java,indent=0,subs="verbatim,quotes"]
----
// Java code
----

XML::
+
[source,xml,indent=0,subs="verbatim,quotes"]
----
<!-- XML code -->
----
======
```

#### 2. Spring Documentation Structure
- `framework-docs/modules/ROOT/pages/` organization
- Transaction docs separation (`declarative/`, `programmatic/`)
- Meaning of `subs="verbatim,quotes"`

#### 3. DCO (Developer Certificate of Origin)
- Using DCO instead of CLA (since January 2025)
- `git commit -s` for automatic Sign-off
- Required for all commits
